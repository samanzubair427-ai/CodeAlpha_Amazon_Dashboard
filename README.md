# CodeAlpha_Amazon_Dashboard
Task 3: Amazon Sales Dashboard
This repository contains the interactive Power BI dashboard designed to visualize Amazon sales performance and key business metrics.

Project Overview:
The goal was to transform raw data into an actionable dashboard, providing insights into category performance, customer ratings, and promotional effectiveness.

Key Features
•	Total Sales Analysis: Visualized total sales against applied discounts to identify revenue drivers.

•	Category Performance: Interactive visuals comparing sales volume and average ratings across different product categories.
•	Dynamic Filtering: Implemented slicers to allow users to filter data by category type.
•	Interactive Visuals: used charts to represents data and analysis

Tools & Techniques
•	Tool:
Power BI Desktop
Power Query (For Data Cleaning)
